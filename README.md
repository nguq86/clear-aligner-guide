## 常见问题

- 在家矫正适合谁，什么情况必须到院？
  轻中度拥挤/间隙、多数排列微调可选在家矫正；若有严重咬合、牙周问题、牙体缺失多或需拔牙配合，优先到院由医生制订方案。

- 第一次用隐形牙套，如何快速上手？
  按品牌指引完成取模或扫描，确认3D方案再启动；前1周严格按时长佩戴，配合咬合块/适应工具，按时更换牙套并记录照片。

- 价格和周期怎么评估是否划算？
  看总费用是否含取模、复诊/远程跟进、精修与保持器；常见疗程约6–12个月，复杂病例更久，选择前先拿到清晰的阶段计划。
